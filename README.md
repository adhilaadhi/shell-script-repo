Shell Scripting Assignment
Project Overview
This project is part of a DevOps shell scripting assignment. The objective is to learn AWS EC2 instance setup, Git version control, Bash scripting, and GitHub repository management by creating a simple EC2 monitoring script.
Objectives
Launch an Ubuntu EC2 instance on AWS.
Install and configure Git.
Initialize a Git repository.
Create a Bash script to monitor system resources.
Push the project to a GitHub repository.
Tasks Completed
Created a free-tier Ubuntu EC2 instance.
Connected to the instance using SSH.
Installed Git on the EC2 instance.
Initialized a Git repository.
Created a monitoring script (monitor.sh).
Displayed running services, CPU usage, memory usage, and disk usage.
Tested the script successfully.
Committed the project and pushed it to GitHub.
